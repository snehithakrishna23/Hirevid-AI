# HireVid AI – Video-First Hiring Platform

🚀 HireVid AI is an AI-powered video-first recruitment platform that transforms traditional hiring by allowing candidates to showcase their skills through video resumes while enabling recruiters to screen, match, and manage talent efficiently.

## 🌐 Live Demo
https://hirevid.vercel.app/

## 📂 GitHub Repository
https://github.com/snehithakrishna23/Hirevid-AI

---

## 📌 Problem Statement

Traditional PDF resumes fail to capture communication skills, confidence, personality, and real-world presence of candidates.

Recruiters spend significant time manually screening resumes without understanding the candidate beyond text.

HireVid AI solves this by introducing:

- Video Resume Profiles
- AI Candidate Matching
- Resume Parsing
- Recruiter Hiring Pipeline
- Virtual Interview Experience
- Candidate Tracking Dashboard

## 🏗 System Workflow

### Candidate Journey
1. Sign up / Login
2. Complete profile
3. Upload resume
4. Record video resume
5. Explore jobs
6. Apply with video profile
7. Track application progress

### Recruiter Journey
1. Sign up / Login
2. Access recruiter dashboard
3. Search candidates
4. Review profiles & video resumes
5. Schedule interviews
6. Move candidates through hiring pipeline
7. Final selection / rejection
